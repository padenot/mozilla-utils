# `mozilla-utils`, a set of scripts to ease the development on mozilla-central

mozattachgdb

> Attach a gdb to a development Firefox that is running.

mozbuild

> Incrementally build a Firefox.

mozcheckdir

> (Utility) Check if we are currently in a mozilla-central directory.

mozmochitest

> Run a Mochitest, maybe inside a debugger.

moznewtree

> Clone a new tree, and setup all the things(tm) automatically.
